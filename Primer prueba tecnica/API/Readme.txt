API

*Pre-requisitos:

Node js     @V15.9.0
React       @17.0.1
puerto disponible 3000

*Instalacion:

*instalacion del servidor (Backend) API

    -Abrir una terminal de comandos desde esta ruta "./API"
    -ingresar a la carpeta "API" (cd API)
    -ejecutar e comando "npm install" para que se instalen las dependencias de produccion y desarrollo
    -ejecutar el comando "npm run start" el cual habilitara el servidor y emitira un mensaje "servidor inializado"
    -El servidor gestiona los datos a travez de un archivo .json ubicado en la carpeta raiz del proyecto

*instalacion del cliente (Front end)

    -Abrir una terminal de comandos desde esta ruta "./API"
    -ingresar a la carpeta "app" (cd app)
    -ejecutar e comando "npm install" para que se instalen las dependencias de produccion y desarrollo
    -ejecutar el comando "npm start" el cual ejecuta la aplicacion cliente y dispone del navegador predeterminado
